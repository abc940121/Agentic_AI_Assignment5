# Query Planner Agent
## Responsibilities
- Expand search terms to account for variations in terminology (Synonym Expansion).
- Select the appropriate knowledge graph indexes to search.

## Strategies
- Maps common terms (e.g., "fee") to technical terms (e.g., "payment", "NTD") and specific values (e.g., "200").
- Broadens the search context based on the identified "aspect" of the question.
